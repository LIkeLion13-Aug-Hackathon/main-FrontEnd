# main-FrontEnd

## 커밋 메시지

| Tag        | Description        |
| ---------- | ------------------ |
| feat       | 새로운 UI/기능 추가       |
| fix        | UI/기능 버그 수정        |
| style      | 스타일 변경 (CSS 수정 등)  |
| layout     | 레이아웃 수정 (구조/정렬 관련) |
| refactor   | 코드 리팩토링 (기능 변화 없음) |
| docs       | 문서 추가, 수정, 삭제      |
| test       | 테스트 코드 추가/수정/삭제    |
| ci         | CI 관련 설정 변경        |
| chore      | 기타 변경사항            |
| remove     | 파일/코드 삭제           |
